# React Redux Webpack Starter
react-webpack-starter를 redux 사용해서 다시 개발한 버전 입니다.

# 빠르게 시작하기
### 선행 준비
* [Node.js](https://nodejs.org)가 설치되어 있어야 합니다.

### 설치
* 프로젝트 디렉토리에서 명령어를 실행합니다.

  ```
  npm install
  ```

### 실행
* 개발용 웹서버를 실행합니다.

  ```
  npm start
  ```

### 확인
* 브라우저로 [http://localhost:8585](http://localhost:8585) 접속합니다.
